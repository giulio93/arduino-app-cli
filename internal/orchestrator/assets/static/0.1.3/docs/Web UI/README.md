# Web UI

This brick allows to host an HTML+JavaScript web application and/or APIs for clients to call.
By default, a web server will be hosted on port 7000.

## Features

- **Host an HTML application**: serves an HTML application built with your favorite JS/TS framework;
- **Host an API**: exposes methods to be called by clients (e.g. a web application);
- **Send messages**: supports pushing messages to the connected clients via WebSockets;
- **Receive messages**: supports listening for messages sent by the connected clients via WebSockets;
- **TLS support**: supports TLS using self-signed certificates.
