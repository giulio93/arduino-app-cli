# Weather Forecasting System

Weather forecast system to display weather information on Led Matrix.

## Overview
This example demonstrates how to use the `weather_forecast` brick to fetch and display weather information on a LED matrix. 
The system retrieves weather data from an API and displays it in a user-friendly format.

## Goals
- Display weather information on a Led Matrix.
- Use the brick `weather_forecast` to fetch weather data from an API.

## Hardware requirements
- No additional hardware is required for this example.

## Instructions
1. Set your city in the sketch file `sketch.ino` by modifying the `city` variable to your desired location.'
   ```cpp
   String city = "Turin";
   ```
2. Run the app.
